Encoded Curtain

A back-end only theme to let users add items via a private Omeka site before going public

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Encoded Curtain.

“Encoded Curtain” is an extremely minimalist Omeka theme meant to temporarily restrict access to your data. Sometimes marking items as “private” isn’t enough; if you’re going to have everything marked as “private” for several months before releasing a site, you might want to let authenticated users add data through the site’s dashboard without your front-end touting zero public items or otherwise looking empty. Maybe you’re developing a theme but want to start adding items through a public URL, for example. Accessing the front-end with the Encoded Curtain theme activated shows you a single, minimal page that lets users with site accounts log into the dashboard and work there as normal.

View this theme on Omeka.org at http://omeka.org/add-ons/themes/encoded-curtain/
